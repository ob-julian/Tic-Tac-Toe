Patch Notes:

  • 5.0.0
    • Changes:
      • There should be no noticeable changes for the user
      • Complet overhaul of the website
        • Added noscript sections to alert the user that JavaScript is required
        • Updated deprecated html tags
        • Made the JavaScript Code more readable (won't show in the browser because it is minified)
        • Faster loading times due to better defer handling of JavaScript and CSS
    • Fixes:
      • Fixed occasional flickering when (re)loading the website
      • Fixed overlapping buttons if unknown hash is used
  • 4.1.0
    • New:
      • The Multiplayer Chat is now End-to-End encrypted. Methode used: RSA

  • 4.0.1.3
    • Fixes:
      • A Chrome Update changed the way it dealt with errors. Therefore some browser-specific functions were not working properly, most noticeable on Chrome Mobile. Those bugs should be fixed now.

  • 4.0.1.2
    • Fixes:
      • If you now use the browser's back button, the website will continue to work instead of displaying multiple subpages at once as before.
        After some tweaking, it even works for Firefox...

  • 4.0.1.1
    • New:
      • When you visit the website for the first time, the light/dark theme is matched to that of the browser
      Fixes:
      • Improved support for Firefox

  • 4.0.1
    • New:
      • Additional tags for a better presentation of the website on social media
      • Small changes that should shorten the response time of the server and allow the website to load faster

  • 4.0.0
    • New:
      • Online multiplayer for experimental mode
    • Changes:
      • Reorganization of the experimantal area

  • 3.3.1
    • New:
      • Animation Speed changer is now implemented. You can change between 0 and 5 Seconds of animation time
    • Changes:
      • Title Headings and the Patch Notes "Icon" are now faded out und in again instead of moving to the side
      • The options in Mobile are now animated too

  • 3.3.0
    • Fixes:
      • Theme changer is visible again in every mode (Desktop Version)
      • Ping works again if you leave the online multiplayer and enter it again
    • New:
      • Patch Notes
      • Chat
      • Animations between all subsites (It is in beta and can still be erratic in some places due to preloading errors)

  • Future plans:
    • Integration of English and German
    • Rework of the medium Bot
    • Increase in the number of supported browsers
    • A whole registration and log-in system (maybe also with google and other support) with xp for matches and daily rewards combined with a level system, achievements, friend system, in turn connected with a challenge option etc.
    • The minimum size of your device is currently 400 x 600 (px). There are plans to further reduce these numbers (low priority)
